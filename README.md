# Stopwatch Web Application

This is a simple and interactive stopwatch web application built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- ⏱ Start the stopwatch  
- ⏸ Pause the stopwatch  
- 🔄 Reset to 00:00:00  
- 🏁 Record lap times
